# 📈 Portfolio Optimization Using Python

## Project Overview

This project implements Modern Portfolio Theory (Markowitz Portfolio Optimization) using Python to construct an optimal investment portfolio. It analyzes historical stock prices, evaluates portfolio risk and return, and identifies the optimal allocation that maximizes the Sharpe Ratio.

---

## Objectives

- Download historical stock market data
- Calculate daily returns
- Analyze risk and expected return
- Build an Efficient Frontier
- Find the Maximum Sharpe Portfolio
- Perform Rolling 30-Day Backtesting
- Compare portfolio performance

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
